Changelog

v1.33 - Attuale
Parser Armi Avanzato.
Miglioramenti:
  • Logica Json Annidato: Il caricamento delle armi da weapons_xx.json ora gestisce gli oggetti annidati dello standard 5e (es. damage.damage_dice e damage.damage_type.name).
  • Nuovo campo Gittata: Aggiunto input specifico per il Raggio (Range) nel modulo delle Armi.
  • Auto-compilazione Dettagliata: Costo, Categoria arma e Danni a due mani (Two-Handed) vengono estratti automaticamente dal file e inseriti nel campo Descrizione. Le Proprietà vengono scompattate dall'array e formattate.
v1.32
Automazione Database Armi.
Miglioramenti:
  • Integrazione menu a tendina nel box Armi per caricare i dati direttamente dalla cartella jsons/.
v1.31
Tracciamento File, Fix i18n Avvio.
Miglioramenti:
  • Visualizzazione del nome file e della data di salvataggio nella barra superiore.
  • Risolto il problema dell'etichetta temporanea "appTitle".
v1.27 - v1.30
Sistema Modificatori, Pesi, Liberazione Layout.
Miglioramenti:
  • Layout a griglia per Box Personaggio. Riordino Box Combattimento.
  • Rimosso `max-width` strutturale. Layout full-screen.
  • Sistema di calcolo "Carico Trasportato" automatico (kg) con penalità dinamiche.
  • Modificatori integrati per Statistiche, TPC e Danni negli equipaggiamenti.
